# Quick Start Guide

## Installation (One Command!)

```bash
cd howdy-gui
sudo ./install.sh
```

That's it! The script will automatically:
- Install Howdy from PPA
- Install all dependencies
- Install Howdy GUI Manager

## First Use

1. **Launch:**
   ```bash
   sudo howdy-gui-manager
   ```
   Or search for "Howdy GUI Manager" in your apps

2. **Setup Camera:**
   - Go to "Camera Settings" tab
   - Select your camera
   - Click "Test Camera"

3. **Add Your Face:**
   - Go to "Face Models" tab
   - Click "Add New Model"
   - Look at camera
   - Done!

4. **Test It:**
   - Click "Test Recognition"
   - Your face should be recognized!
